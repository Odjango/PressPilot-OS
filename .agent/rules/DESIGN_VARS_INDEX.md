# PRESSPILOT: DESIGN DNA INDEX (PROVEN CORES)
This file serves as the "Pre-Trained Memory" for the UI/UX Pro Max module. 
Always reference these structural and aesthetic boundaries when generating new themes.

---

## 1. BLOCKBASE (The Minimalist Blueprint)
- **Vibe**: The "Blank Canvas." Professional, clean, and invisible framework.
- **Typography**: Uses system fonts by default to prioritize speed and native feel.
- **Layout**: Strict adherence to standard WordPress template hierarchy.
- **Best For**: "Corporate" or "Utility" verticals where content is the only focus.

## 2. FROST (The High-End Agency)
- **Vibe**: Sophisticated, ultra-clean, and "Masterpiece" quality.
- **Palette**: High-contrast (Base #ffffff, Contrast #000000) with a signature "Primary Blue".
- **Typography**: Features "Fluid Typography" that scales smoothly across devices.
- **Best For**: "Portfolio" or "Agency" verticals needing a polished, expensive look.

## 3. OLLIE (The Modern Startup)
- **Vibe**: Vibrant, mobile-first, and energetic.
- **Palette**: Multi-color presets (Neon, Teal, Pink) for high brand personality.
- **Typography**: Uses modern sans-serif scales for a tech-focused aesthetic.
- **Best For**: "Startup" or "SaaS" verticals where speed and "pop" are key.

## 4. SPECTRA-ONE (The Performance Pro)
- **Vibe**: Sleek and versatile with professional-grade layouts.
- **Palette**: 9 built-in variations (Mango, Aquamarine) focusing on accessible contrast.
- **Typography**: Heavily integrated with Google Fonts (1,500+ options).
- **Best For**: "Business" or "Blog" verticals requiring deep visual customization.

## 5. TOVE (The Creative Vertical)
- **Vibe**: Fun, colorful, and highly specialized for niche industries.
- **Specialty**: Built-in logic for restaurant menus, opening hours, and table bookings.
- **Layout**: Playful with multi-column grid patterns for visual storytelling.
- **Best For**: "Restaurant" and "Cafe" verticals (The immediate roadmap item).

## 6. TWENTY TWENTY-FOUR (The Multipurpose Gold Standard)
- **Vibe**: Elegant, sophisticated, and universally "Unbreakable".
- **Palette**: Light default (#f9f9f9) with refined shades like Onyx and Ember.
- **Typography**: Cardo (Headings) for "Old Style" refinement + Inter (Body) for readability.
- **Best For**: Any vertical where a "Timeless" and accessible design is required.
