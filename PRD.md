1.  **DO NOT SIMULATE:** Do not pretend to work. Write actual code.
2.  **NO TECH DEBT:** If you change a component, do not break its existing styling.
3.  **ASK BEFORE DESTRUCTIVE ACTIONS:** If you plan to delete a folder or rewrite a core config (like `next.config.js`), ask for permission first.
