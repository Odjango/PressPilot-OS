
This structure integrates all FSE best practices we confirmed:

Pure PHP for Content: functions.php handles only page creation and site settings.

Pure HTML Blocks for Layout: All templates use stable block markup, eliminating recovery errors.

Correct Menu Order: The navigation is hardcoded in the header for universal consistency across all generated themes.

Below are the contents for a theme named presspilot-gold-test. You can assemble these files into a folder named presspilot-gold-test and compress it into a .zip to test.

3. functions.php (Root of Theme)
This is the cleanest version of your PHP code. It contains ONLY the page creation logic (presspilot_create_required_pages) and is completely free of the conflicting menu functions.

PHP
<?php
/**
 * PressPilot Gold Test helpers.
 */

if (!defined('ABSPATH')) {
    exit;
}

/**
 * Universal function to create required pages on theme activation.
 */
function presspilot_create_required_pages()
{
    // Universal set of pages to create: [ 'Page Title' => 'page-slug' ]
    $pages_to_create = array(
        'Home' => 'home',
        'Menu' => 'menu',
        'About' => 'about',
        'Services' => 'services',
        'Blog' => 'blog',
        'Contact' => 'contact'
    );

    foreach ($pages_to_create as $page_title => $page_slug) {
        // Check if the page already exists (by slug)
        if (!get_page_by_path($page_slug)) {
            $page_data = array(
                'post_type' => 'page',
                'post_title' => $page_title,
                'post_name' => $page_slug,
                'post_content' => 'This is the ' . $page_title . ' page. Edit this content via Pages > ' . $page_title,
                'post_status' => 'publish',
                'post_author' => 1
            );
            wp_insert_post($page_data);
        }
    }

    // Set 'Home' as the static front page and 'Blog' as the posts page.
    $home_page = get_page_by_path('home');
    $blog_page = get_page_by_path('blog');

    if ($home_page) {
        update_option('page_on_front', $home_page->ID);
        update_option('show_on_front', 'page');
    }
    if ($blog_page) {
        update_option('page_for_posts', $blog_page->ID);
    }
}
// Hook runs when ANY PressPilot theme is activated.
add_action('after_switch_theme', 'presspilot_create_required_pages');

// Note: Removed the presspilot_register_menus and presspilot_setup_fse_navigation functions.
