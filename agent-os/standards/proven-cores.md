# THE LAW
1. SOURCE OF TRUTH: The 5 allowed theme templates are in `/proven-cores/`.
2. NEVER create a theme from scratch.
3. STARTING A TASK: Always COPY a core folder to your working directory first.
4. IMMUTABLE: The `functions.php` structure in the cores is SACRED. Do not change the PHP logic, only the CSS/Design.
# BRANDING REQUIREMENTS (MANDATORY)
5. FOOTER COMPLIANCE (MANDATORY):
   - **The Credit Line:** You must REPLACE the theme's copyright text with this EXACT format:
     `© [Year] [Site Title]. All rights reserved. Powered by PressPilot.`
     *Link 'PressPilot' to: https://presspilot.app*
   - **Social Icons:** You must ensure the footer contains a Social Icons block (`core/social-links`).
   - **Design Integrity:** Do NOT force specific background colors (like orange) or layouts. Adapt the text and icons to fit the *current theme's* design aesthetic (Dark, Light, Minimal, etc.).
