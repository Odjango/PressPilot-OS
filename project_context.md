# PROJECT CONTEXT: PressPilot (AI WordPress Theme Generator)

## 1. PROJECT MISSION
PressPilot is a SaaS platform that generates complete, valid WordPress websites in 90 seconds. 
- **Core Mechanism:** User Input (Biz Info + Logo) -> AI Content Generation -> Pattern Injection -> Live WordPress Site.
- **Critical Rule:** We DO NOT generate themes file-by-file. We use a "Site Assembly" method that injects block patterns into a stable base theme (Astra/Neve).

## 2. ARCHITECTURE & FILE MAP (SOURCE OF TRUTH)
The logic is strictly modular. The Agent must strictly use these files:

### A. The "Brain" (Content)
- **File:** `modules/content-generator.php`
- **Class:** `PressPilot_Content_Generator`
- **Role:** Calls OpenAI (GPT-3.5) to generate text/JSON based on the patterns in `patterns/{type}.json`.
- **Agent Rule:** Never hallucinate content prompts. Use the `generate()` method in this class.

### B. The "Hands" (Assembly)
- **File:** `modules/site-assembler.php`
- **Class:** `PressPilot_Site_Assembler`
- **Role:** Takes the JSON content and "bakes" it into WordPress pages (`wp_insert_post`) using Gutenberg block comments.
- **Agent Rule:** If I ask to fix a page layout, check `generate_modern_home_html()` in this file. Do not write raw HTML.

### C. The "Eyes" (Design)
- **File:** `modules/color-extractor.php`
- **Class:** `PressPilot_Color_Extractor`
- **Role:** Extracts Hex codes from the uploaded logo.
- **File:** `modules/theme-matcher.php`
- **Role:** Selects the installed parent theme (Astra/Neve/Blocksy).

### D. The "Bridge" (Agent Control)
- **File:** `modules/agent-bridge.php` & `modules/mcp-server.php`
- **Role:** Exposes internal logic to the Antigravity Agent via REST API (`/wp-json/presspilot/v1/mcp`).
- **Tools:** The Agent can trigger actions via JSON commands like `content.page.create` or `design.colors.set`.

## 3. "NO HALLUCINATION" PROTOCOLS
1.  **Block Patterns Only:** All page output must match the structure in `patterns/home-modern.json`.
    - Correct Format: ``
    - Incorrect Format: `<div class="hero">...</div>`
2.  **Respect the Bakery:** We are running inside a live WordPress instance (Coolify). Do not suggest `npm run build`. We operate via PHP and the WP REST API.
3.  **Authentication:** All external requests must be signed with the Token generated by `agent-auth.php`.
